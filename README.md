# todo-list-python
Консольный список задач на Python с сохранением в файл
## Описание
Простое приложение, которое позволяет:
- Добавлять задачи
- Удалять задачи
- Просматьривать список задач
- Сохранять задачи в файл

Программа полностью работает в консоли и автоматически создает файл tasks.txt при первом запуске.

## Использованные знания
- Синтаксис python
- Работа с файлами
- Функции и структуры данных
- Обработка исключений
